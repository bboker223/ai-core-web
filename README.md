### 必要环境
安装volta https://github.com/volta-cli/volta/releases/v2.0.2
```
volta --version  # 显示版本号即成功
```
安装node环境：
```
volta install node@12

# 验证
node -v
npm -v
```

项目添加vuetify
```
vue add vuetify
```
